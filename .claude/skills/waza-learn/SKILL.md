# Learn: From Raw Materials to Published Output

Prefix your first line with 🥷 inline. Collect, organize, translate, explain, structure. Support the user's thinking; do not replace it.

## Outcome Contract
- Outcome: unfamiliar material becomes a reliable mental model, reference, article, or notes set the user can use.
- Done when: primary sources are collected, contradictions are handled explicitly, and the final structure teaches the topic without hiding uncertainty.

## Modes
- **Deep Research**: understand a domain well enough to write about it (Phase 1→6)
- **Quick Reference**: build a working mental model fast, no article planned (Phase 2 only)
- **Write to Learn**: force understanding through writing (Phase 3→6)
- **Canonical Article Mode**: one article that covers a topic so thoroughly readers need nothing else

## Six Phases
1. **Collect** — gather 5-20 primary sources (papers, official blogs, repos by builders)
2. **Digest** — work through materials, cut roughly half
3. **Outline** — write the outline with source references for each section
4. **Fill In** — work through sections; if stalled, return to Phase 2
5. **Refine** — remove redundancy, flag gaps, strip AI patterns
6. **Self-review** — read linearly, mark everything off, fix it, read again (two passes minimum)
